# Digital-Inventory-Manager
A Python-based terminal application designed to track stock levels, manage product quantities, and organize inventory efficiently.
# Digital Inventory Manager (Python)

This is my first Python project.
It is a console-based Digital Inventory Manager.

## Features
- Show inventory items
- Add new items
- Remove items
- Exit program

## Concepts Used
- Python lists
- Loops (while, for)
- Conditional statements
- User input/output

## How to Run
1. Install Python
2. Run the file:
   python inventory_manager.py
